Wisielec (Hangman)

JavaScript - React (Router, Redux)

Prosta aplikacja zainspirowana szkolną grą w wisielca.

W obecnej wersji umożliwia odgadywanie hasła, które jest jednym słowem.

Umożliwia tworzenie haseł w 3 językach. Zawiera klawiatury dla:

  • polskiego
  • angielskiego
  • rosyjskiego

Planuję przetłumaczenie interfejsu na wyżej wymienione języki za pomocą react-i18next.

Gra stworzona przy pomocy Create React App.
